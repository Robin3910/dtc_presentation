# 品牌素材审核规范 v2.0

> 本文件由品牌运营团队维护，AI 将依据此规范对达人素材进行逐项审核。

---

## 一、图片规范

- [ ] **竞品 logo 禁止出现**：华为、OPPO、小米、三星、荣耀、realme 等竞品 logo 不得出现在画面中
- [ ] **主色调限制**：禁止以红色为主色调，品牌主色为蓝绿渐变
- [ ] **禁止外部水印**：图片不得带有非品牌方水印（达人个人水印属于外部水印）
- [ ] **分辨率要求**：图片分辨率不低于 1080×1080px
- [ ] **禁止拉伸变形**：图片不得经过拉伸、压缩变形或过度美颜处理
- [ ] **禁止模糊马赛克**：图片内容清晰，不得有大面积马赛克或模糊区域
- [ ] **品牌 logo 要求**：若涉及品牌元素，需使用品牌 logo，不得使用竞品 logo

## 二、文案规范

- [ ] **禁止极限词**：不得出现「第一」「最好」「国家级」「全网最低」「顶级」等绝对化用语
- [ ] **禁止虚假承诺**：不得出现「保证美白」「一定有效」「无效退款」等无法兑现的承诺（已附资质证明的除外）
- [ ] **禁止夸大功效**：文案描述的功效需与产品备案功效一致，不得夸大
- [ ] **禁止联系方式**：不得出现微信号、手机号、网址链接、二维码
- [ ] **禁止对比贬低**：不得出现贬低竞品或夸大自身对比优势的描述
- [ ] **合规用语参考**：可使用「用户真实反馈」「因人而异」「肌肤水润透亮」等表达

## 三、视频规范

- [ ] **时长限制**：视频时长不超过 60 秒（联盟平台硬性限制）
- [ ] **品牌元素要求**：视频前 3 秒内必须出现品牌元素（logo、产品包装等）
- [ ] **背景音乐合规**：背景音乐需为免费商用音乐或已购买版权，不得使用未授权热门歌曲
- [ ] **字幕规范**：如有字幕，需与配音内容一致，不得出现错别字

## 四、综合要求

- [ ] **版权合规**：图片和视频素材需为达人原创或已获得合法授权，不得侵犯第三方版权
- [ ] **未成年人保护**：内容不得涉及未成年人不当使用场景
- [ ] **违禁内容**：不得出现涉及政治、色情、暴力、赌博等违禁内容
