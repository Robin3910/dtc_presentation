PROBLEM · 痛点分析

素材审核有哪些痛点？

1、人工审核容易因为疲劳、情绪导致标准不一致，且规则众多时容易遗漏。

2、同时，素材审核本身属于比较固定且机械化的流程，机器人来做更合适。

EVOLUTION · 演变历程

AI 图片审核的
演变与突破

传统方案

机器学习模型

  • 需要大批量图片数据
  • 专门训练专用模型
  • 规则修改需重新训练
  • 周期长、成本高
现在方案

AI 大模型 + 提示词

  • 直接调用，无需训练
  • 修改提示词即可调整规则
  • 当天就能跑起来
  • 成本低、迭代快

提示词稳定后，每天把审核任务丢给 AI，等 AI 跑完再看结果

PRICING · 计价模式

模型 Token 的计价模式
两种主流方式

方式一：月度订阅制

包月限量

  • 按订阅套餐，每月支付固定费用
  • 常见于知名平台（如 Cursor、Claude Code）
  • 适合有稳定高频需求的用户
  • 优点：预测性好，预算可控
方式二：按用量计费

按 token 数量计价

  • 根据实际消耗的 token 数量计费（如 1M tokens 若干元）
  • 适用于 API 调用和自托管部署场景
  • 灵活精准，适合不确定/低频使用
  • 优点：按需付费，不浪费配额

选择建议： 如果你每天高频写作或开发，优选月度订阅；如果只是偶尔调用，则选择按量计费更灵活实惠。
